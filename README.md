# Football Transfer Market Valuation: QA & Prediction Pipeline## Project OverviewA data engineering and machine learning pipeline that estimates the "Fair Market Value" of European footballers. This project automates the data collection from FBref/Transfermarkt, enforces data integrity using **Great Expectations**, and trains a regression model to identify market inefficiencies.## Key Features* **Automated Scraping:** Python scripts to aggregate stats from disparate sources.* **Data Quality Assurance:** Automated validation suite (Great Expectations) to catch anomalies (e.g., negative values, unrealistic ages) before ingestion.* **Market Modeling:** Linear Regression analysis to quantify the "English Premier League Tax."## Tech Stack* **Python 3.10+*** **Data Engineering:** Pandas, SQLAlchemy* **QA/Testing:** Great Expectations* **Modeling:** Scikit-Learn* **Visualization:** Streamlit## Setup & Reproducibility1. Clone the repo2. Install dependencies: `pip install -r requirements.txt`3. Run the scraper: `python src/scraper.py`4. Run QA checks: `python tests/validation_suite.py`
